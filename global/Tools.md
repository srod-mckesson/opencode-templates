# Tools

Useful tools I use with opencode.

## rtk

- If installed, it runs as an opencode hook and cannot be bypassed by agents
- It summarizes output to save tokens and cost
- Repository: https://github.com/rtk-ai/rtk

## Ponytail

- Prefer using it for over-engineering reduction and minimal implementations
- Repository: https://github.com/DietrichGebert/ponytail

## Roborev

- Use for commit review workflow when installed
- Repository: https://github.com/kenn-io/roborev
